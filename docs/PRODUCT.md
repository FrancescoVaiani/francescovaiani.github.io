# Product

This project is a bilingual (EN/IT) personal website for Francesco "Sito" Vaiani, designed to work as both a modern portfolio and an always-updated CV.

## High-Level Features

- Clear Product Manager-first positioning with full-stack background context
- Structured sections: hero, profile, experience, product work, approach, media, skills, contact
- English/Italian language switch
- Desktop-first responsive experience with accessible navigation
- Single-source content rendered for both screen and print/PDF (`@media print`)

## Design System

The implementation must respect the existing design system and token source in `/design-system/tokens`.
